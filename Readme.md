# Adoptify

Adoptify ist eine kleine Webanwendung, die als Lernprojekt entstanden ist. Ziel ist eine Übersicht und Verwaltung von Tieren, die „adoptiert“ oder ausgeliehen werden können.

## Features

* Tierprofile mit Bildern
* Kategorien/Tags
* Grundlegende Listen- und Detailansichten

## Tech-Stack

* HTML, CSS, JavaScript
* Backend Python, Flask

## Status

Dieses Projekt befindet sich im Aufbau und dient als Übungsprojekt für Frontend-Strukturierung und Web-Komponenten.
